Smart Lighting Controllers

Type 1 – Lighting Automation Controller & UI –

  • This replaces the mechanical tic-tac switches with electronic touch switches for the user to operate manually or remotely and a CPU fits in the conceal box to control the appliances. This can control the lights, fans, Air – conditioners, dimmable lights, curtains, water heaters, TV or projector lifts. This type of system lets you control almost anything and you can consider this for your home or office lighting.
  • They are good looking and match with the interiors if required.
  • This technology is software based and that is the biggest advantage of it from the maintenance point of view.
  • This system is retrofit ready in most of the cases. i.e. no need to change the existing wiring.

Type 2 – Lighting Automation Controller –

  • This works with the mechanical switches in two modes for lighting control. The hardware goes behind the switch plate and lighting can be operated using Android and iOS applications on mobile or tablets.
  • This system is retrofit ready in most of the cases. i.e. no need to change the existing wiring.
